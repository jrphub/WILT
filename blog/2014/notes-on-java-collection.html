Complexity

  • ArrayList : Adding/Removing an element at rear end : O(1)
  • ArrayList : Adding/Removing an element at any position : O(n)
  • ArrayList : Searching an element : O(n)
  • HashSet (Storage method is Hash Table) : Adding/Removing/contains an element : O(1)
  • TreeSet (Storage method is red black Tree) : Adding/Removing/contains an element : O(logn)
  • LinkedHashSet ( implemented as a hash table with a linked list running through it) : Adding/Removing/contains an element : O(1)
  • HashMap : Adding/removing an element : O(1)
  • TreeMap : Adding/Removing an element : O(logn)
